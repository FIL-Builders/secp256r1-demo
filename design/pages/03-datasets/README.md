# Datasets

<!-- markdownlint-disable MD013 MD012 -->

Purpose: chain-backed dataset list for the selected root wallet and network.

The page should show:

- active network
- dataset count
- provider
- dataset ID
- client dataset ID
- metadata
- pieces count
- payment rail state
- proof status
- last activity

Local storage must not be required for this page to populate.

Default view should use consumer-style dataset cards or soft rows. Protocol-level fields belong in detail or Advanced views.
