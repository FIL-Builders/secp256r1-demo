# Settings Prompt

<!-- markdownlint-disable MD013 MD012 -->

```text
Design a Settings page for Synapse.

Use `images/settings-v1.png` as the visual reference when available. It is a tall scrollable settings reference, not a fixed single-viewport constraint.

Include:
- default network preference
- preferred authorization mode
- passkey credential labels
- local cache management
- developer mode toggle
- privacy controls
- advanced developer verification checks entry

Make clear that clearing local cache removes local preferences and labels but does not delete chain-backed datasets, pieces, payments, or passkey authorizations.

The navbar and sidebar network state must stay synchronized with the default network selection.

Use a compact dashboard settings layout.
```
